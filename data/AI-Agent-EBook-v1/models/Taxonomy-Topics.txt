TIER 2: AUDIENCE SEGMENTS
--------------------------
Match content depth and vocabulary to the primary audience.

Executive (C-suite, VP):
- Focus: strategy, ROI, risk, competitive advantage
- Avoid: technical implementation details
- Length: shorter chapters, more frameworks and summaries

Manager (Director, Sr. Manager):
- Focus: implementation planning, team enablement, vendor selection
- Balance: strategic context + practical how-to
- Length: standard chapter length

Practitioner (Analyst, Engineer, Specialist):
- Focus: how-to, tools, code patterns, technical depth
- Include: specific tools, workflows, technical examples
- Length: can go longer with more detail

TIER 2: CONTENT FORMATS BY USE CASE
-------------------------------------
Thought Leadership:     opinion + framework + evidence. No how-to steps.
Practical Guide:        step-by-step. Include templates, checklists.
Case Study:             problem → approach → outcome. Use real numbers.
Reference / Playbook:   comprehensive. Designed to be bookmarked.
Quick-Start Guide:      minimal. 5 steps. Get to value in 30 minutes.
